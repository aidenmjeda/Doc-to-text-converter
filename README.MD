# Document to Text Converter (DOCX + PDF)

This project converts `.docx` and `.pdf` files into plain text so they can be used with Large Language Models (LLMs) such as Ollama, OpenAI, or any RAG pipeline.

---

## 🚀 Features

- Extract text from **.docx files**
- Extract text from **.pdf files**
- Outputs clean `.txt` files
- Ready for LLM / AI pipelines
- Simple and lightweight
